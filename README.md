# Space Cadet Pinball for RCade

A port of [Space Cadet Pinball](https://github.com/alula/SpaceCadetPinball) for the [RCade](https://github.com/fcjr/RCade) arcade platform.

## About

This project brings the classic 3D Pinball - Space Cadet game to RCade, based on the reverse-engineered decompilation by alula.
